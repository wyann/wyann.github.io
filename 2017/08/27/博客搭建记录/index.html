博客搭建记录

发表于 2017-08-27 | 分类于 博客搭建

这篇文章主要记录通过hexo搭建个人博客网站的一个过程，以便后续博客网站迁移维护。

前置条件

安装Node.js、Git

博客初始化

安装hexo 

npm install hexo-cli -g

新建博客目录 

mkdir blog

初始化博客目录 

hexo init blog  
npm install

启动服务端进行预览 

hexo s

在GitHub中部署

git配置

git初始化配置 

git config --global user.name 'your name'  
git config --global user.email 'your email'

生成公钥私钥 

ssh-keygen -C 'your email' -t rsa

将公钥配置到github中

github博客库配置

在github中new respository,库名称必须为’login user name’.github.io
点击设置，GitHub Pages选择一个主题生成github博客网站

hexo-github关联配置

在博客根目录下_config.yml文件中增加如下配置： 

deploy:
  type: git
  repo: git@github.com:wyann/wyann.github.io.git
  branch: master

安装部署插件 

npm install hexo-deployer-git --save

部署到github

hexo g
hexo d

博客美化
